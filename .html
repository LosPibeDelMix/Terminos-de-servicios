Términos de Servicio

1. Introducción

Al invitar o utilizar LosPibesDelMix Bot (en adelante, el Bot), aceptás los términos y condiciones que figuran en este documento. Si no estás de acuerdo con alguno de los puntos, no uses el Bot.

2. Descripción del servicio

El Bot provee funciones dentro de servidores de Discord, incluyendo (pero no limitado a): utilidades, comandos con prefijo, moderación básica (kick/ban/purge), mensajes automáticos de bienvenida y comandos informativos. Las funcionalidades pueden variar con actualizaciones.

3. Uso permitido y prohibido

Está prohibido usar el Bot para:

  • Spam, raids, automatizaciones destructivas o flood.
  • Difundir contenidos ilegales, peligrosos o que promuevan daño.
  • Acoso, doxxing o amenazas.
  • Suplantación o manipulación de identidades.
  • Violación de los Términos de Servicio de Discord.

4. Limitación de responsabilidad

El Bot se proporciona "tal cual" y no garantiza disponibilidad continua ni ausencia de errores. Los desarrolladores no son responsables por daños directos o indirectos, pérdidas de datos ni consecuencias derivadas del uso o mal uso del Bot.

5. Cambios en los términos

Estos términos pueden modificarse. Las actualizaciones se publicarán en la URL donde esté alojado este documento y en el servidor oficial cuando corresponda.

Política de Privacidad

1. Alcance

Esta Política describe qué datos procesa el Bot y cómo se tratan dentro del entorno de Discord. El Bot opera únicamente dentro de la plataforma Discord y no recopila datos fuera de ella.

2. Datos procesados

El Bot puede procesar temporalmente los siguientes datos para cumplir sus funciones:

  • ID de usuario, nombre de usuario y apodos.
  • Avatar (URL proporcionada por Discord).
  • Roles asignados en el servidor.
  • Mensajes cuando son necesarios para ejecutar comandos o moderación.
  • Datos básicos del servidor: ID, nombre, canales y roles.

El Bot NO almacena mensajes privados, historiales de chat ni información sensible fuera de Discord.

3. Almacenamiento y retención

El Bot no guarda información permanentemente en bases de datos externas ni vende datos. El procesamiento ocurre en memoria durante la ejecución del Bot y se descarta al apagarse.

4. Compartición de datos

No se comparte, vende ni distribuye información a terceros. Los únicos servicios implicados son la API oficial de Discord y el servidor donde se ejecute el Bot (VPS o hosting elegido por el operador).

5. Seguridad

Se aplican medidas razonables de seguridad: protección del token, permisos mínimos necesarios y control de acceso al código. Sin embargo, ningún sistema puede garantizar seguridad absoluta.

6. Permisos solicitados

El Bot puede solicitar permisos como leer y enviar mensajes, administrar mensajes, gestionar miembros (kick/ban) y ver canales. Estos permisos se usan exclusivamente para las funciones del Bot.

7. Menores

El Bot no está diseñado para recopilar información adicional sobre menores y respeta las políticas de Discord relacionadas con la edad de los usuarios.

8. Cambios en la política

Esta política puede actualizarse y la fecha de última modificación aparecerá al inicio del documento.
